Component ExceptionsHandler
To handle exceptions at runtime and fix errors at debug time.
Component Level: Low
Category: Software-User Components
Find errors at debug time.

This component includes an implementation of HardFault interruption than helps you to find the code line that causes a fault. Then you can fix it.

See Typical Usage for more information.

Handle errors at runtime.

If you want to prevent errors and handle them at runtime then use try/catch instuctions. You also can create your own exceptions with throw instruction.

See Typical Usage for more information.
