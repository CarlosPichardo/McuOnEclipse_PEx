Component LEDReverseBias
Driver for a LED used as a photodiode as well for communication.
Component Level: Low
Category: CPU External Devices-Display
Methods:
(Methods are user-callable functions/subroutines intended for the component functions control. Please see the Embedded Components page for more information.)
  • On - This turns the LED on.

      ANSIC prototype: void On(void)

  • Off - This turns the LED off.

      ANSIC prototype: void Off(void)

  • Neg - This negates/toggles the LED

      ANSIC prototype: void Neg(void)

  • Get - This returns logical 1 in case the LED is on, 0 otherwise.

      ANSIC prototype: byte Get(void)
    • Return value:byte - Status of the LED (on or off)

  • Put - Turns the LED on or off.

      ANSIC prototype: void Put(byte val)
    • val:byte - value determin if the LED has to be on or off.

  • StartSafeReverseBias - Puts the LED in reverse bias mode and starts discharigng (port set as input). This method stores the actual port state.

      ANSIC prototype: void StartSafeReverseBias(void *portBackup)
    • portBackup: Pointer to byte - Pointer to backup structure for port status (to be restored later). It points to a %'ModuleName'%.PortStatus structure to be allocated by the caller.

  • GetCathodeVal - Returns the cathode value.

      ANSIC prototype: ComponentName_SensorValType GetCathodeVal(void)
    • Return value:ComponentName_SensorValType - Error code

  • StopSafeReverseBias - Restores the port to its original settings using the saved port state.

      ANSIC prototype: void StopSafeReverseBias(void * portBackup)
    • portBackup: Pointer to byte - Pointer to backup structure for port status to be restored. It points to a %'ModuleName'%.PortStatus structure to be allocated by the caller.

  • StartReverseBiasLedOn - Puts the LED (which is in ON state) in reverse bias mode and starts discharging (port set as input).

      ANSIC prototype: void StartReverseBiasLedOn(void)

  • StopReverseBiasLedOff - Restores the port to output and LED OFF.

      ANSIC prototype: void StopReverseBiasLedOff(void)

  • StopReverseBiasLedOn - Restores the port to output and LED ON.

      ANSIC prototype: void StopReverseBiasLedOn(void)

  • StartReverseBiasLedOff - Puts the LED (which is in OFF state) in reverse bias mode and starts discharging (port set as input).

      ANSIC prototype: void StartReverseBiasLedOff(void)


Note: Some methods can be implemented as macros.
