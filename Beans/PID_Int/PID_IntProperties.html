Component PID_Int
PID Controller using integer arithmetic.
Component Level: High
Category: SW-Control
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component name - Name of the component.
  • Variable for controller set point - If the set point value shall be reflected as global variable (for visualization or debugging).
  • P - Proportional - The P (Proportional) portion of the controller.
    The following items are available only if the group is enabled (the value is "Enabled"):
    • constant - If factor/divisior shall use constant values or be variable. If values are non-constant, you can change them as variables at runtime (using Visualization tool). This is useful for finding the right values, but impacts performance.
    • Kp Factor - The Kp multiplication factor. For efficiency reason this should be a power of 2.
    • Kp Divisor - The Kp divisor value for Kp. For efficiency reason this should be a power of 2.
  • I - Integral - The I (Integral) portion of the controller.
    The following items are available only if the group is enabled (the value is "Enabled"):
    • constant - If factor/divisior shall use constant values or be variable. If values are non-constant, you can change them as variables at runtime (using Visualization tool). This is useful for finding the right values, but impacts performance.
    • Ki Factor - The Ki multiplication factor. For efficiency reason this should be a power of 2.
    • Ki Divisor - The Ki divisor value for Ki. For efficiency reason this should be a power of 2.
    • Anti Wind-Up - Anti-Windup value for the integral part.
      The following items are available only if the group is enabled (the value is "Enabled"):
      • constant - If factor/divisior shall use constant values or be variable. If values are non-constant, you can change them as variables at runtime (using Visualization tool). This is useful for finding the right values, but impacts performance.
      • Value - Cap value of integral portion
  • D - Differential - The P (Proportional) portion of the controller.
    The following items are available only if the group is enabled (the value is "Enabled"):
    • constant - If factor/divisior shall use constant values or be variable. If values are non-constant, you can change them as variables at runtime (using Visualization tool). This is useful for finding the right values, but impacts performance.
    • Kd Factor - The Kd multiplication factor. For efficiency reason this should be a power of 2.
    • Kd Divisor - The Kd divisor value for Kd. For efficiency reason this should be a power of 2.
  • System - Interface to the system.
    • Sensor - The sensor which reports the speed
    • Actuator - The motor to be used
