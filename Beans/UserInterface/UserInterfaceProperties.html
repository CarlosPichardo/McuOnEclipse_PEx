Component UserInterface
User Interface for LCD displays
Component Level: High
Category: CPU External Devices-Display
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component name - Name of the component.
  • Fixed Settings - Configure some settings as fixed in order to reduce the RAM needs
    • Animated buttons - If set to yes, then the UI expects both a 'press' and a 'release' call to ClickWindowElement. this is useful e.g. for Button transitions.
    • Selection - If selection of elements shall be enabled. You may disable this to save the needed RAM and ROM amount.
    • Text Foreground Color - Uses a predefined color for the text (forground)
      The following items are available only if the group is enabled (the value is "Enabled"):
      • Text Foreground Color - To reduce RAM usage, a fixed text forground color can be used.

        There are 16 options:
        • BLACK: Black color
        • WHITE: White color
        • RED: Red color
        • BRIGHT RED: Bright red color
        • DARK RED: Dark red color
        • GREEN: Green color
        • BRIGHT GREEN: Bright green color
        • DARK GREEN: Dark green color
        • BLUE: Blue color
        • BRIGHT BLUE: Bright blue color
        • DARK BLUE: Dark blue color
        • YELLOW: Yellow color
        • BRIGHT YELLOW: Bright yellow color
        • ORANGE: Orange color
        • GREY: Grey color
        • BRIGHT GREY: Bright grey

    • Text Font - Uses a predefined text font
