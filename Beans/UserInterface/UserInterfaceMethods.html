  • ChangeText - Changes the text of the element. The method will erase the existing text and replaces it with the new text, and copies the new text into the existing buffer (that's why you need to provide the size of the existing buffer as well).

      ANSIC prototype: byte ChangeText(ComponentName_Window *window, ComponentName_Element *elem, size_t dstTextSize, byte *txt)
    • window: Pointer to ComponentName_Window - Pointer to window
    • elem: Pointer to ComponentName_Element - Pointer to the element.
    • dstTextSize:word - Size in bytes of existing text.
    • txt: Pointer to byte - Pointer to new text. Note that this text has to be different from the element text shown, and this text will become the new element text.
    • Return value:byte - error code

  • AssignText - Changes the text of the element. The method will use the pointer to the new text provided. The caller is responsible to deallocate the previous text if necessary.

    • txt: Pointer to byte - Pointer to new text. Note that this text has to be different from the element text shown, and this text will become the new element text.
    • Return value:byte - error code
