DCF-77 Receiver
Component Level: Low

Methods:
(Methods are user-callable functions/subroutines intended for the component functions control. Please see the Embedded Components page for more information.)
  • Start - Starts the sampling of the time signal.

      ANSIC prototype: void Start(void)

  • Stop - Stops the sampling of the time signal.

      ANSIC prototype: void Stop(void)

  • IsPresent - Function checks if a DCF receiver is present. As the signal is open drain with a pull-up, we can detect a receiver in case we get a low signal. This function checks if there is a receiver present for at most ˜200 ms as a positive pulse may last at least 200 ms.

      ANSIC prototype: bool IsPresent(void)
    • Return value:bool - Error code

  • IsScanning - Returns true if we are scanning the signal, false otherwise.

      ANSIC prototype: bool IsScanning(void)
    • Return value:bool - Error code

  • ScanBit - Needs to be called in a period way with a period as specified by the bean properties.

      ANSIC prototype: void ScanBit(void)


Note: Some methods can be implemented as macros.
