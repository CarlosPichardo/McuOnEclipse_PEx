Component SSD1289
Display driver for the Solomon Systech SSD1289 display
Component Level: Low
Category: CPU External Devices-Display
Application Notes:
(Examples, notes and hints for the component usage.)
Below are the settings for several board & LCD combinations. Note that the Rev.0 board does not have the micro-SD card with the MCF51JM128 CPU on the TWR-LCD.

TWR-LCD Rev.A board jumper settings (standalone mode with onboard MCF51JM128)
SW1 switch settings (default settings, Standalone JM mode, JM128 with SPI mode, JM128 reads from microSD):
1: Off (PS2, 4-wire SPI mode)
2: On (PS1, 4-wire SPI mode)
3: Off (JM_ELE disabled, MISO controlled by JM128)
4: Off (EuSD not routed to the elevator)
5: On (SPI_SEL, using ELE_CS1)
6: On (TP_SEL, touch pannel selection, on is let the JM128 read the touchpanel)
7: On (BL_CTRL, backlight on)
8: Off (ELE_PWM0, BUZZER not on Elevator)

SW5 switch settings (default all off, controlling the 4 lines to the touch screen):
1: Off (XPLS not routed to elevator)
2: Off (YMNS not routed to elevator)
3: Off (XMNS not routed to elevator)
4: Off (YPLS not routed to elevator)

TWR-LCD Rev.A board jumper settings (LCD controlled by the TWR CPU, e.g. MCF51CN128)
SW1 switch settings (16bit parallel mode, LCD, sounder and touchscreen controlled by TWR MCU):
1: On (PS2, 16bit parallel mode)
2: Off (PS1, 16bit parallel mode)
3: Off (JM_ELE disabled, MISO controlled by JM128)
4: On (EuSD routed to the elevator)
5: On (SPI_SEL, using ELE_CS1)
6: Off (TP_SEL, touch pannel selection, TWR MCU is reading touch panel)
7: On (BL_CTRL, backlight on)
8: On (ELE_PWM0, BUZZER routed to Elevator)

SW5 switch settings (default all off, controlling the 4 lines to the touch screen):
1: On (XPLS routed to elevator)
2: On (YMNS routed to elevator)
3: On (XMNS routed to elevator)
4: On (YPLS routed to elevator)

TWR-LCD Rev.0 board jumper settings
Display Protocol communication settings:
PS2 PS1
On Off : 16bit 6800 (16bit mini-FlexBus) (default)
On On : 3-wire SPI
Off On : 4-wire SPI
On On : 8bit 6800

Using the TWR-LCD Rev.0 Module with SPI and TWR-MCF51CN
  • TWR-MCF51CN jumper settings:
    • ANA-ISO: 1-2, 3-4, 5-6 (PTE0,E1,E2) as used for accelerometer
    • ANA-ISO: remove 7-8 (PTD3) which is used for the touchscreen (X+)
    • ANA-ISO: remove 9-10 (PTD2) which is used for touchscreen (Y-)
    • ANA-ISO: remove 11-12 (PTD1) which is used for touchscreen (X-)
    • ANA-ISO: remove 13-14 (PTD0) which is used for touchscreen (Y+)
    • X_ISO, Y_ISO and Z_ISO can be installed if accelerometer is used by the application
    • LED3_ISO and LED4_ISO can be installed if used by the application
    • LED1-ISO: remove jumper as signal TPM1_CH0 is used for LCD Buzzer
    • LED2-ISO: remove jumper as signal PTG5 is used for LCD right navigation key
    • LATCH_SEL: jumper on 2-3 (enable address latch for mini-FlexBus)
  • TWR-LCD Rev.0 jumper settings:
    • BL-ISO: jumper installed (provide power to backlight)
    • CS-SEL: jumper on 2-3(using SPI_CS)
    • SPI-CS: jumper on 1-2(using SPI0_CS0)
    • PS2: jumper removed
    • PS1: jumper installed (4-wire SPI)
  • SPI settings:
    • PTC4/SPI0_CS0: CS (Chip Select CS0)
    • PTC5/SPI0_MOSI: MOSI (Master Out, Slave In)
    • PTC7/SPI0_CLK: CLK (Clock)
    • PTF3/FB_CS0_B: D/C (Data/Command)
