Component GFont
Graphic Font
Component Level: High

Methods:
(Methods are user-callable functions/subroutines intended for the component functions control. Please see the Embedded Components page for more information.)
  • GetFontChar - Returns for a given character the corresponding font bitmap.

      ANSIC prototype: PGFONT_CharInfo GetFontChar(byte ch)
    • ch:byte - The character for that a bitmap is required.
    • Return value:PGFONT_CharInfo - Character font bitmap

  • GetBoxHeight - Function to get the height of the bounding box.

      ANSIC prototype: byte GetBoxHeight(void)
    • Return value:byte - Height of bounding box

  • GetFont - Returns callbacks for fonts to be used by the font driver.

      ANSIC prototype: PGFONT_Callbacks GetFont(void)
    • Return value:PGFONT_Callbacks - Font callbacks.

  • GetUnderlineBoxHeight - Returns the size of the underline box height

      ANSIC prototype: byte GetUnderlineBoxHeight(void)
    • Return value:byte - Error code

  • GetLineSpaceHeight - Returns the size of the line space height

      ANSIC prototype: byte GetLineSpaceHeight(void)
    • Return value:byte - Error code


Note: Some methods can be implemented as macros.
