Component FAT_FileSystem
FAT file system module
Component Level: High
Category: SW-Memory Manager
This bean implements a FAT file system.
The bean creates the sources of the file system driver:

The sources can be retrieved from http://elm-chan.org/fsw_e.html

Thanks to ChaN for his great work!

Important note: make sure your SD card is formatted properly.

The other thing to consider: the driver is using some amount of stack. So if you are running into problems, increasing the stack to say 0x100 bytes might help.
