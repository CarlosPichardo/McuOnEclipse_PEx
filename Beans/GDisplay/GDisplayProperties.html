Component GDisplay
Graphic Display Driver
Component Level: High
Category: CPU External Devices-Display
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component name - Name of the component.
  • Inverted Pixels - Using this options, the pixels on the display will be inverted (e.g. what would be black will be white, and vice versa).
  • Memory Buffer - Enabled if the display driver is using a memory buffer.
    The following items are available only if the group is enabled (the value is "Enabled"):
    • Orientation - Orientation of the display

      There are 4 options:
      • Landscape: Display in landscape orientation
      • Landscape180: Display in landscape orientation, rotatied by 180 degree
      • Portrait: Display in portrait orientation
      • Portrait180: Display in portrait orientation, rotated by 180 degrees

  • Clear screen on Init - If the screen shall be clear on Init
  • Hardware - Interface to the display hardware
    • Display - Interface to the display
  • Watchdog - Display operations may take a long time. If using a watchdog, then the display driver may need to kick the dog.
    The following items are available only if the group is enabled (the value is "Enabled"):
    • Watchdog - Interface to watchdog
  • RTOS - Interface to an RTOS for mutual exclusive access to the display
    The following items are available only if the group is enabled (the value is "Enabled"):
    • RTOS - Interface to RTOS for mutual exclusion to display
