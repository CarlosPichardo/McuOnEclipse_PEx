Text Widget
Component Level: High
Category: CPU External Devices-Display
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component name - Name of the component.
  • Selection - If text selection is enabled or not
  • Defaults - Configures the widget defaults
    • Font - Font for the CheckBox Label
    • Text - Default text
    • Text Foreground Color - Default text foreground color to be used

      There are 16 options:
      • BLACK: Black color
      • WHITE: White color
      • RED: Red color
      • BRIGHT RED: Bright red color
      • DARK RED: Dark red color
      • GREEN: Green color
      • BRIGHT GREEN: Bright green color
      • DARK GREEN: Dark green color
      • BLUE: Blue color
      • BRIGHT BLUE: Bright blue color
      • DARK BLUE: Dark blue color
      • YELLOW: Yellow color
      • BRIGHT YELLOW: Bright yellow color
      • ORANGE: Orange color
      • GREY: Grey color
      • BRIGHT GREY: Bright grey

    • Text Background Color - Default text foreground color to be used

      There are 16 options:
      • BLACK: Black color
      • WHITE: White color
      • RED: Red color
      • BRIGHT RED: Bright red color
      • DARK RED: Dark red color
      • GREEN: Green color
      • BRIGHT GREEN: Bright green color
      • DARK GREEN: Dark green color
      • BLUE: Blue color
      • BRIGHT BLUE: Bright blue color
      • DARK BLUE: Dark blue color
      • YELLOW: Yellow color
      • BRIGHT YELLOW: Bright yellow color
      • ORANGE: Orange color
      • GREY: Grey color
      • BRIGHT GREY: Bright grey

  • System - Interface to shared system modules
    • User Interface - Interface to User Interface
    • Font Display - Font Display to be used
    • Utility - Interface to utility functions
