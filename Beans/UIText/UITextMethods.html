Bean UIText
Text Widget
Component Level: High
Category: CPU External Devices-Display
Methods:
(Methods are user-callable functions/subroutines intended for the component functions control. Please see the Embedded Components page for more information.)
  • Create - Creates a text element

      ANSIC prototype: byte Create(ComponentName_Element *parent, ComponentName_TextWidget *txtw, ComponentName_PixelDim x, ComponentName_PixelDim y, ComponentName_PixelDim width, ComponentName_PixelDim height)
    • parent: Pointer to ComponentName_Element - Pointer to parent element. If NULL, then the widget will not be added to the list of widgets.
    • txtw: Pointer to ComponentName_TextWidget - Pointer to text widget
    • x:ComponentName_PixelDim - x position
    • y:ComponentName_PixelDim - y position
    • width:ComponentName_PixelDim - width of element
    • height:ComponentName_PixelDim - height of element
    • Return value:byte - Error code

  • ChangeText - Changes the text of the element. The method will erase the existing text and replaces it with the new text, and copies the new text into the existing buffer (that's why you need to provide the size of the existing buffer as well).

      ANSIC prototype: byte ChangeText(ComponentName_TextWidget *txtw, word dstTextSize, unsigned char *txt)
    • txtw: Pointer to ComponentName_TextWidget - Pointer to the text widget
    • dstTextSize:word - Size in bytes of existing text.
    • txt: Pointer to char - Pointer to new text. Note that this text has to be different from the element text shown, and this text will become the new element text.
    • Return value:byte - Error code

  • SetFont - Sets the font of the widget.

      ANSIC prototype: byte SetFont(ComponentName_TextWidget *widget, ComponentName_Font *font)
    • widget: Pointer to ComponentName_TextWidget - Pointer to text widget
    • font: Pointer to ComponentName_Font - Pointer to font
    • Return value:byte - Error code

  • GetFont - Sets the font of the widget.

      ANSIC prototype: ComponentName_PFont GetFont(ComponentName_TextWidget *widget)
    • widget: Pointer to ComponentName_TextWidget - Pointer to text widget
    • Return value:ComponentName_PFont - Error code

  • SetText - Changes the text of the element. The method will use the pointer to the new text provided. The caller is responsible to deallocate the previous text if necessary.

      ANSIC prototype: byte SetText(ComponentName_TextWidget *txtw, unsigned char *txt)
    • txtw: Pointer to ComponentName_TextWidget - Pointer to the text widget
    • txt: Pointer to char - Pointer to new text. Note that this text has to be different from the element text shown, and this text will become the new element text.
    • Return value:byte - Error code

  • GetText - Returns a pointer to the text

      ANSIC prototype: ComponentName_PText GetText(ComponentName_TextWidget *widget)
    • widget: Pointer to ComponentName_TextWidget - Pointer to widget
    • Return value:ComponentName_PText - pointer to text string

  • GetTextBoxWidth - Returns the width of the text based on font information. As the text might be multiline with \r, it will return the width of the longest line.

      ANSIC prototype: ComponentName_PixelDim GetTextBoxWidth(ComponentName_TextWidget *widget)
    • widget: Pointer to ComponentName_TextWidget - Pointer to widget
    • Return value:ComponentName_PixelDim - Error code

  • GetTextBoxHeight - Returns the height of the text based on font information. As the text might be multiline with \r, it will return the total height, not only of one line.

      ANSIC prototype: ComponentName_PixelDim GetTextBoxHeight(ComponentName_TextWidget *widget)
    • widget: Pointer to ComponentName_TextWidget - Pointer to widget
    • Return value:ComponentName_PixelDim - Error code

  • GetFontHeight - Returns the height of the text based on font information, for a single line of text.

      ANSIC prototype: ComponentName_PixelDim GetFontHeight(ComponentName_TextWidget *widget)
    • widget: Pointer to ComponentName_TextWidget - Pointer to widget
    • Return value:ComponentName_PixelDim - Error code

  • Resize - Resize the widget area based on the text in it.

      ANSIC prototype: byte Resize(ComponentName_TextWidget *widget)
    • widget: Pointer to ComponentName_TextWidget - Pointer to text widget
    • Return value:byte - error code

  • SetBgColor - Change the background color of the text.

      ANSIC prototype: byte SetBgColor(ComponentName_TextWidget *txtw, ComponentName_PixelColor bgColor)
    • txtw: Pointer to ComponentName_TextWidget - Pointer to text widget
    • bgColor:ComponentName_PixelColor - text background color
    • Return value:byte - Error code

  • SetFgColor - Change the foreground color of the text.

      ANSIC prototype: byte SetFgColor(ComponentName_TextWidget *txtw, ComponentName_PixelColor fgColor)
    • txtw: Pointer to ComponentName_TextWidget - Pointer to text widget
    • fgColor:ComponentName_PixelColor - text color
    • Return value:byte - Error code

  • SetUserMsgHandler - Set additional user message handler.

      ANSIC prototype: void SetUserMsgHandler(ComponentName_TextWidget *textW, %@UI@'ModuleName'%.MsgHandler handler)
    • textW: Pointer to ComponentName_TextWidget - Pointer to text widget
    • handler:byte - Callback to be used


Note: Some methods can be implemented as macros.
