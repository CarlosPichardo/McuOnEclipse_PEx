Bean UIButton
Button Widget
Component Level: High
Category: CPU External Devices-Display
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component name - Name of the component.
  • Selection - If button can be selected.
  • Animated buttons - If the button should change the shape for press and release.
  • Button Defaults - Defaults which are used
    • TextWidget - Interface to text
    • Text - Default Text
    • Background Color - Defines the color of the button background. Default is BRIGHT_BLUE.

      There are 16 options:
      • BLACK: Black color
      • WHITE: White color
      • RED: Red color
      • BRIGHT RED: Bright red color
      • DARK RED: Dark red color
      • GREEN: Green color
      • BRIGHT GREEN: Bright green color
      • DARK GREEN: Dark green color
      • BLUE: Blue color
      • BRIGHT BLUE: Bright blue color
      • DARK BLUE: Dark blue color
      • YELLOW: Yellow color
      • BRIGHT YELLOW: Bright yellow color
      • ORANGE: Orange color
      • GREY: Grey color
      • BRIGHT GREY: Bright grey

    • Top Line Color - Defines the color of the button top part. Default is BLACK.

      There are 16 options:
      • BLACK: Black color
      • WHITE: White color
      • RED: Red color
      • BRIGHT RED: Bright red color
      • DARK RED: Dark red color
      • GREEN: Green color
      • BRIGHT GREEN: Bright green color
      • DARK GREEN: Dark green color
      • BLUE: Blue color
      • BRIGHT BLUE: Bright blue color
      • DARK BLUE: Dark blue color
      • YELLOW: Yellow color
      • BRIGHT YELLOW: Bright yellow color
      • ORANGE: Orange color
      • GREY: Grey color
      • BRIGHT GREY: Bright grey

    • Bottom Line Color - Defines the color of the button bottom part. Default is WHITE.

      There are 16 options:
      • BLACK: Black color
      • WHITE: White color
      • RED: Red color
      • BRIGHT RED: Bright red color
      • DARK RED: Dark red color
      • GREEN: Green color
      • BRIGHT GREEN: Bright green color
      • DARK GREEN: Dark green color
      • BLUE: Blue color
      • BRIGHT BLUE: Bright blue color
      • DARK BLUE: Dark blue color
      • YELLOW: Yellow color
      • BRIGHT YELLOW: Bright yellow color
      • ORANGE: Orange color
      • GREY: Grey color
      • BRIGHT GREY: Bright grey

    • Line Width - Defines the width/thickness of the button border line of the button
    • Text Border Width - Defines the width of space around the button text
    • Default Text Foreground Color - Default text foreground color to be used. Default is BLACK.

      There are 16 options:
      • BLACK: Black color
      • WHITE: White color
      • RED: Red color
      • BRIGHT RED: Bright red color
      • DARK RED: Dark red color
      • GREEN: Green color
      • BRIGHT GREEN: Bright green color

      • GREY: Grey color
      • BRIGHT GREY: Bright grey

  • System - Interface to shared system modules
    • User Interface - Interface to UserInterface
