Component OneWire
Component for the One Wire Protocol
Component Level: High
Category: SW-Communication
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component Name - Name of the component.
  • Data Pin I/O - Link component (for details about settings see <a href="../../DOCs/InheritanceAndSharing.html">Component Inheritance &amp; Component Sharing</a>).
  • TimerUnit - Link component (for details about settings see <a href="../../DOCs/InheritanceAndSharing.html">Component Inheritance &amp; Component Sharing</a>).
  • Connection time settings -
    • Connection Mode -

      There are 2 options:
      • Master - One slave: Master, one slave
      • Master - Multiple devices: Master, multiple devices

    • A: Write 1 Low time (us) - Period that master holds the DQ signal low to send a bit '1'. Default is 6
    • B: Write 1 High time (us) - Period that master holds the DQ signal low to send a bit '1'. Default is 64
    • C: Write 0 Low time (us) - Period that master holds the DQ signal low to send a bit '0'. Default is 60
    • D: Write 0 High time (us) - Period that master holds the DQ signal low to send a bit '0'. Default is 10
    • E: delay time before read (us) - Time to wait after releasing signal until to read the bit. Default is 9. Shorten this if read of bit is too late.
    • F: After read delay time - Delay time after reading the bit until the next slot.
    • H: Reset time (us) - Duration of the reset pulse that the master send to slave devices. Default 480
    • I: Device response time (us) - Time that master waits before read, after send the reset pulse. Default 70.
    • Total slot time (us) - Total communication slot time. Default 100.
  • Buffers -
    • Output - Output buffer
    • Input - Input buffer
    • Time - Buffer for time
    • Program - Buffer for program
  • CriticalSection - Link component (for details about settings see <a href="../../DOCs/InheritanceAndSharing.html">Component Inheritance &amp; Component Sharing</a>).
