Component VS1053
Component for the VLSI VS1053 Audio Codec Device
Component Level: Low
Category: SW-User Components
Methods:
(Methods are user-callable functions/subroutines intended for the component functions control. Please see the Embedded Components page for more information.)
  • ActivateSPI - Called to activate SPI bus.

      ANSIC prototype: byte ActivateSPI(void)
    • Return value:byte - Error code

  • DeactivateSPI - Method called to deactivate SPI bus

      ANSIC prototype: byte DeactivateSPI(void)
    • Return value:byte - Error code

  • IsReady - Returns TRUE if device is ready.

      ANSIC prototype: bool IsReady(void)
    • Return value:bool - TRUE if ready, FALSE otherwise

  • WriteRegister - Writes a device register

      ANSIC prototype: byte WriteRegister(byte reg, word value)
    • reg:byte - register to write
    • value:word - Value to write
    • Return value:byte - Error code

  • ReadRegister - Reads a device register

      ANSIC prototype: byte ReadRegister(byte reg, word *value)
    • reg:byte - Register to read
    • value: Pointer to word - Pointer to where to write the value read
    • Return value:byte - Error code

  • WriteWRAM - Writes a device WRAM

      ANSIC prototype: byte WriteWRAM(word addr, word value)
    • addr:word - WRAM address
    • value:word - Value to write
    • Return value:byte - Error code

  • ReadWRAM - Read from device WRAM

      ANSIC prototype: byte ReadWRAM(word addr, word *value)
    • addr:word - WRAM address
    • value: Pointer to word - Pointer to value to return
    • Return value:byte - Error code

  • SendValues - Sends a number of values

      ANSIC prototype: byte SendValues(word nofValues, byte value)
    • nofValues:word - Number of values to send
    • value:byte - Values to send
    • Return value:byte - Error code

  • StartSong - Starts a song with sending 10 zeroes

      ANSIC prototype: byte StartSong(void)
    • Return value:byte - Error code

  • TerminateStreams - Stops a currently playing stream

      ANSIC prototype: byte TerminateStreams(void)
    • Return value:byte - Error code

  • SetVolume - Sets the volume

      ANSIC prototype: byte SetVolume(byte left, byte right)
    • left:byte - zero is max volume, 0xfe is silence, 0xff is analog power down
    • right:byte - zero is max volume, 0xfe is silence, 0xff is analog power down
    • Return value:byte - Error code

  • SendData - Sends a block of data to the device

      ANSIC prototype: byte SendData(byte *data, size_t dataSize)
    • data: Pointer to byte - Pointer to data
    • dataSize:byte - Size of the data in bytes
    • Return value:byte - Error code

  • LoadUserCode - Loads a user code or plugin into the device.

      ANSIC prototype: byte LoadUserCode(const uint16_t *data, word nofWords)
    • data: Pointer to word - Pointer to data of the plugin
    • nofWords:word - Number of 16bit words of data
    • Return value:byte - Error code

  • LoadRealtimeMidiPlugin - Loads the realtime MIDI plugin to the device

      ANSIC prototype: byte LoadRealtimeMidiPlugin(void)
    • Return value:byte - Error code

  • MIDI_Talk - Plays a MIDI note

      ANSIC prototype: byte MIDI_Talk(byte cmd, byte data1, byte data2)
    • cmd:byte - command byte
    • data1:byte - data1 byte
    • data2:byte - data2 byte
    • Return value:byte - Error code

  • MIDI_NoteOn - Send a MIDI note-on message, like pressing a piano key.

      ANSIC prototype: byte MIDI_NoteOn(byte channel, byte note, byte velocity)
    • channel:byte - Channel is from 0-15
    • note:byte - Note to be played
    • velocity:byte - pressing velocity of key
    • Return value:byte - Error code

  • MIDI_NoteOff - Send a MIDI note-off message, like releasing a piano key.

      ANSIC prototype: byte MIDI_NoteOff(byte channel, byte note, byte velocity)
    • channel:byte - Channel is from 0-15
    • note:byte - Note to be played
    • velocity:byte - releasing velocity of key
    • Return value:byte - Error code

  • MIDI_SetVolume - Sets the volume for the channel

      ANSIC prototype: byte MIDI_SetVolume(byte channel, byte volume)
    • channel:byte - Channel (0..15)
    • volume:byte - volume, 0 (silence) to 127 (max volume)
    • Return value:byte - Error code

  • MIDI_SetBank - Sets the bank for the channel

      ANSIC prototype: byte MIDI_SetBank(byte channel, byte bank)
    • channel:byte - Channel
    • bank:byte - bank (0 is default, 0x78 and 0x7f is drums, 0x79 melodic)
    • Return value:byte - Error code

  • MIDI_SetInstrument - Sets the instrument for the channel

      ANSIC prototype: byte MIDI_SetInstrument(byte channel, byte instrument)
    • channel:byte - Channel
    • instrument:byte - instrument number (0..127)
    • Return value:byte - Error code

  • MIDI_SetPan - Sets the pan (stereo position) for the channel

      ANSIC prototype: byte MIDI_SetPan(byte channel, byte pan)
    • channel:byte - Channel
    • pan:byte - pan (stereo position), where 64 is middle. Allowed values: 0..127
    • Return value:byte - Error code

  • MIDI_AllSoundOff - Turns off all oscillators for the channel

      ANSIC prototype: byte MIDI_AllSoundOff(byte channel)
    • channel:byte - Channel
    • Return value:byte - Error code

  • ParseCommand - Shell Command Line parser. This method is enabled/disabled depending on if you have the Shell enabled/disabled in the properties.

      ANSIC prototype: byte ParseCommand(const unsigned char *cmd, bool *handled, const %@Shell@'ModuleName'%.StdIOType *io)
    • cmd:byte - Pointer to command string
    • handled: Pointer to bool - Pointer to variable which tells if the command has been handled or not
    • io: Pointer to byte - Pointer to I/O structure
    • Return value:byte - Error code

  • Deinit - Driver deinitialization routine

      ANSIC prototype: byte Deinit(void)
    • Return value:byte - Error code

  • Init - Driver initialization routine

      ANSIC prototype: byte Init(void)
    • Return value:byte - Error code


Note: Some methods can be implemented as macros.
