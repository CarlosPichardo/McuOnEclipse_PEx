Component SSD1351
Display driver for the Solomon Systech SSD1351 display (e.g. on Hexiwear)
Component Level: Low
Category: CPU External Devices-Display
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component name - Name of the component.
  • Type - Display type used

    There are 2 options:
    • 96x96: e.g. display used in Hexiwear
    • 128x128: e.g. Adafruit 1.5" Color OLED (#1431)

  • Orientation - Orientation of the display.

    There are 4 options:
    • landscape: Landscape format
    • landscape180: Landscape, rotated by 180 degree
    • portrait: Portrait, landscape rotated right by 90 degree
    • portrait180: Portrait, landscape rotated left by 90 degree

  • Width - Width of the display in pixels.
  • Height - The height of the display in pixels.
  • Bytes in rows - If bytes are organized in rows or columns.
  • Bytes in x direction - If bytes are increasing in x direction (from left to right). Otherwise bytes are increasing in y direction (from top to down).
  • MSB first - If the Most Significant bit/pixel comes first in the byte.
  • Bits per pixel - Number of bits per pixel. 1 for B/W displays.
  • Window capability - If the display is able to use a local window on the device to write data.
  • Display Memory Write - If we can directly write into the display memory
  • Display Memory Read - If we can read from the display memory. For this you need the RD pin from the display. Note that you cannot read from the display in serial mode.
  • Use RAM Buffer - If enabled, the driver is able to use a RAM buffer for the display memory. Use the UpdatFull() to transfer the data to the display.
  • Clear display in Init() - If the display driver shall clear the display background during initialization.
  • Initialize on Init - If the driver shall be initialized during Init
  • Settings - Settings to configure the display hardware. Use the shell to find the optimal settings.
    • Contrast A - Contrast A value (Command 0xC1), default is 0xC8 for Adafruit SSD1351 display
    • Contrast B - Contrast B value (Command 0xC1), default is 0x80 for Adafruit SSD1351 display
    • Contrast C - Contrast C value (Command 0xC1), default is 0xC8 for Adafruit SSD1351 display
    • Contrast Master - Contrast Master value (Command 0xC7), default is 0x0F for Adafruit SSD1351 display
  • HW - Group controlling the hardware settings.
    • HW SPI Delay (us) - Optional Hardware SPI delay time after sending a character, in microsconds
    • SW SPI - Group to control the Software SPI.
      The following items are available only if the group is enabled (the value is "Enabled"):
      • SW SPI - Software SPI interface
    • LDD HW SPI - Using LDD (Logical Device Driver) SPI driver
      The following items are available only if the group is enabled (the value is "Enabled"):
      • HW SPI - SPI communication interface
    • HW SPI - This sets up a Hardware SPI (non-LDD) communication
      The following items are available only if the group is enabled (the value is "Enabled"):
      • HW SPI - Hardware SPI interface
      • Baud Rate Mode - If the HW SPI channel is used with different devices and baud rate, this number specifies the baud rate to be used. 0 is the initial baud rate.
    • Slave Select - if we are directly controlling the slave select pin (chip select)
      The following items are available only if the group is enabled (the value is "Enabled"):
      • SCE - SPI Chip Enable/Select signal (low active)
    • Reset - Enable this group if a normal reset pin can be used. Otherwise you need to fill in the information for a manual reset pin handling
      The following items are available only if the group is enabled (the value is "Enabled"):
      • Reset - Display reset signal (low active)
      • D_C - Data/Command mode select pin. Low level is command mode.
    • Power - Enable this group if there is a dedicated hardware power up/down pin
      The following items are available only if the group is enabled (the value is "Enabled"):
      • Power - Interface to Power Pin
  • System - Group controlling the system settings.
    • Wait - Interface for busy waiting.
    • SDK - Link to SDK used
  • Shell - Optional command line shell support
    The following items are available only if the group is enabled (the value is "Enabled"):
    • Shell - Interface to shell
    • Utility - Link to utility functions
