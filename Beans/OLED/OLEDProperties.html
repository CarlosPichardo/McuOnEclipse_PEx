Component OLED
Densitron OLED Display
Component Level: Low
Category: CPU External Devices-Display
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component name - Name of the component.
  • Number of colors - How much colors are required. The less the fewer resources will be used.

    There are 2 options:
    • 256: 256 colors
    • 65k: 65k colors

  • Width - Width of the display in pixels. Our display is 64x96
  • Height - Height of the display in pixels
  • Bytes in rows - If bytes in display memory are organized in rows.
  • MSB first - If the Most Significant Bit comes first in a display byte.
  • Bits per pixel -

    There are 3 options:
    • 1 (B/W): Black/White mode
    • 8 (256 Colors): 256 Color mode
    • 16 (65k Colors): 65k Color mode

  • Hardware - This group controls the hardware connection.
    • 6800 mode -
    • BS1 - Configuration of the BS1 pin.
      The following items are available only if the group is enabled (the value is "Enabled"):
      • BS1 pin -
    • BS2 - Configuration of the BS1 pin.
      The following items are available only if the group is enabled (the value is "Enabled"):
      • BS2 pin -
    • RES pin - Power reset pin for display controller and driver. When the pin is low, the display chip will be reset.
    • DC pin - Data/Command control pin. When the pin is pulled high, then the input at D7..D0 is treated as data. When the pin is pulled low, the input at D7..D0 will be used as command.
    • RW pin - This pin is used as Read/Write selection input. This pin will be set high for read mode and pulled low for write mode.
    • E pin - This pin is used as Enable signal. Read/Write operation is initiated when this pin is pulled high and the CS pin is pulled low.
    • CS pin - Chip select. In order to communicate with the display this pin is pulled down.
    • Data pins - This pins are used for the data and commands.
  • System - This group defines the system interface.
    • Wait Interface - Interface to waiting methods.
